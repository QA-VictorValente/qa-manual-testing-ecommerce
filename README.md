# QA Manual Testing - E-commerce Project

Este projeto simula testes manuais em uma aplicação de e-commerce, com foco em qualidade de software, validação de regras de negócio e documentação de testes.

## 🎯 Objetivo
Demonstrar habilidades em:
- Planejamento de testes
- Criação de casos de teste
- Execução de testes manuais
- Registro de bugs
- Documentação de evidências

## 🧪 Escopo dos Testes
- Login de usuário
- Cadastro de usuário
- Pesquisa de produtos
- Adição ao carrinho
- Finalização de compra

## 📁 Estrutura
- Test Plan
- Test Cases
- Bug Reports
- Evidences
